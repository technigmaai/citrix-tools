# Citrix Tools

A collection of PowerShell tools for managing Citrix environments.

## Tools

- **Profile Management Tools**: A collection of PowerShell tools for managing user profiles in Citrix environments, including migrations and maintenance operations.
- **VHD(X) Cleanup Tool**: Scan and clean VHD/VHDX images by mounting them and deleting specified paths.
